# 🎓🐾 SmartPet Productivity

## 🎓🐾 Um sistema de produtividade gamificado para estudantes e pessoas disciplinadas.

- Projeto em desenvolvimento que integra foco, disciplina e evolução digital através de um pet virtual.

- O objetivo é transformar produtividade real (Pomodoro, estudo e treino) em progresso visível dentro do sistema.

__🚀 Funcionalidades atuais__

1. Sistema de pet com fome, energia e humor

2. Decaimento automático de status

3. Sistema de XP e level

4. Pomodoro integrado (25 minutos)

5. Contador de minutos produtivos

6. Persistência com localStorage

__🎯 Objetivo__

- Criar uma ferramenta útil para o dia a dia que incentive disciplina e consistência através de gamificação.

__📌 Status__

- Interface ainda em fase inicial (protótipo funcional).
- Melhorias visuais e novas funcionalidades serão implementadas progressivamente.
